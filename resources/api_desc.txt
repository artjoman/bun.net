REST

/party
GET -> retrieves all as partyList.json
POST -> adds new, returns HTTP status

/party/{partyId}
GET -> get specific party as party.json
POST -> update, returns HTTP status

/party/{partyId}/bun
GET -> retrieves all for partyId as bunList.json
POST -> adds new, returns HTTP status
